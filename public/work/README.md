# Work PDFs

Drop your project PDFs here. Filename must match the `pdfSlug` you set in `src/data/work.ts`.

**Example:**  
- File: `branding-2024.pdf`  
- In `work.ts`: `pdfSlug: 'branding-2024'`

Then add or edit the project entry in `src/data/work.ts` (title, description, slug, pdfSlug).

If you want it to show under a specific section, also set `category` in `src/data/work.ts`.
